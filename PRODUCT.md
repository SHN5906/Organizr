# Organizr — PRODUCT.md

register: product

## Product Purpose

Outil solo de planification : missions de montage vidéo et production de sites
web pour des clients. Deux gestes : capturer une mission en moins de 15 s
(raccourci « n »), scanner le mois en 5 s sur un calendrier monochrome.

## Users

Un seul utilisateur : Sohan, monteur vidéo / créateur de sites freelance.
Usage le matin et entre deux exports, bureau lumineux, desktop d'abord,
mobile en appoint. Toujours en cours de tâche : l'outil doit disparaître.

## Brand & Tone

Papeterie de travail : planning papier, encre noire, filets fins, blanc
généreux. Sobre, précis, silencieux. Français, tutoiement, phrases courtes.

## Anti-references

- Le « template admin SaaS » : sidebar + cards + chips colorées + gradients.
- Le dashboard IA générique : glassmorphism, ombres colorées, emojis décoratifs.
- Le brutalisme terminal (monospace partout, all-caps agressif) — c'est le
  deuxième réflexe du monochrome, pas notre registre.

## Strategic principles

1. 100 % monochrome : noir, blanc, gris. Les états se distinguent par la
   FORME (rempli / creux / pointillé), jamais par la teinte.
2. La densité sert le scan : hiérarchie typographique nette, pas de décoration.
3. Le calendrier est l'écran signature ; le reste s'efface derrière lui.
